# 🚀 Object-Oriented Programming Project | Digital Notice Board System
This repository contains the complete C++ source code and documentation for the Digital Notice Board & Announcement System, developed to implement and practice core Object-Oriented Programming (OOP) concepts.

## 📌 Project Overview
This system is designed to replace traditional paper-based notice boards with a digital platform where faculty and administrators can post announcements, and students can view relevant notices efficiently.


## 🔑 Key Features
* **Role-based System:** Separate access and functionalities for Admin, Faculty, and Student users.
* **CRUD Operations:** Ability to create, edit, delete, and display announcements in real-time.
* **Categorization:** Notices are organized into distinct categories (Academic, Events, General, Emergency).
* **File Handling:** Integrated permanent data storage to retain announcements after closing the system.
* **Clean Class Structure:** Built using robust and reusable OOP principles.


## 🧠 OOP Concepts Used
* **Classes & Objects:** Modeled real-world entities like Users, Announcements, and Notice Boards.
* **Encapsulation:** Secured sensitive data fields using private modifiers and accessors (getters/setters).
* **Inheritance:** Implemented a user hierarchy where Admin and Faculty inherit from a base User class.
* **Polymorphic Logic:** Allowed dynamic handling of different notice categories and user privileges.
* **File Handling:** Used `fstream` for permanent data storage and management.
* **Modular Code:** Structured the application for high reusability and clean code organization.
